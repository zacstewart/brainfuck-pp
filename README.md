# brainfuck++

A toy brainfuck interpreter written in C++. I wrote this as a learning exercise
to understand pointers. You can read about it [here][blog-post].

## To Build

You need GCC.

    make

## Run It

```
./brainfuck ">++++++++[<+++++++++>-]<.>>+>+>++>[-]+<[>[->+<<++++>]<<]>.+++++++..+++.>
>+++++++.<<<[[-]<[-]>]<+++++++++++++++.>>.+++.------.--------.>>+.>++++."
Hello World!
```
